Becoming a Web Developer

Project Description

Welcome to becoming a web developer. The goal of this website is to give you, yes YOU a first hand top down look at the core aspects of entering this challenging and satisfying field. Here you will find information on the different jobs a web developer may undertake as well as the core programming/scripting languages that one would need to know to enter this field. I will also provide information on various online tools and web hosting options that can make your entry into this field a lot smoother. First we will go through the different fields of work a web developer may want to pursue. Next we will examine the different computer scripting and programming languages a web developer will need to know in order to be successful. Moving onto tools, that is different online resources that can be very helpful to web developers. Finally we will look at web hosting, and the different types that there are. You will be pleased to find that one can enter the web developer field with only access to a computer. The many tools and online resources are available free of charge to get your education and self learning under way.
